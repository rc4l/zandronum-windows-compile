I compiled these OpenSSL static libraries via Perl to save you time. If you prefer compiling it yourself, you can 
delete the contents of this OpenSSL folder and run the script. It will take a while but that'll
install Perl and compile the OpenSSL binaries for you to use with Zandronum inside deps. ~ Cata
